# Política de Privacidade do Atlas T9

**Última atualização:** Julho de 2026

## 1. Coleta de informações

O Atlas T9 não coleta informações pessoais dos usuários.

O aplicativo não exige cadastro, criação de conta ou login.

## 2. Processamento local

Todo o processamento das sugestões de digitação ocorre exclusivamente no dispositivo do usuário.

O texto digitado nunca é enviado para servidores externos.

## 3. Dados armazenados localmente

Para melhorar as sugestões, o Atlas T9 pode armazenar localmente:

- palavras adicionadas pelo usuário;
- frequência de uso das palavras;
- relações entre palavras (bigramas e trigramas);
- preferências do aplicativo.

Esses dados permanecem apenas no dispositivo.

## 4. Exclusão dos dados

O usuário pode apagar todos os dados de aprendizado e palavras armazenadas a qualquer momento nas configurações do aplicativo.

## 5. Compartilhamento

O Atlas T9 não vende, compartilha ou transmite dados pessoais ou dados de digitação para terceiros.

## 6. Conexão com a Internet

Nesta versão, o Atlas T9 funciona integralmente de forma offline para as funções de digitação.

Caso futuras versões adicionem funcionalidades online, esta política será atualizada.
